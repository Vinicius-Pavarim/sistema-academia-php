# Sistema de Academia em PHP

Sistema web desenvolvido para fins acadêmicos, simulando a gestão básica de uma academia.  
A aplicação permite autenticação de usuários e o gerenciamento de usuários, alunos e professores, utilizando PHP, MySQL e CSS em ambiente local.

---

## Funcionalidades

- Login e logout de usuários
- Controle de acesso por autenticação
- Cadastro, listagem e exclusão de:
  - Usuários
  - Alunos
  - Professores
- Interface estilizada com CSS
- Persistência de dados utilizando MySQL
- Estrutura organizada com includes para autenticação e layout

---

## Tecnologias Utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- phpMyAdmin
- Ambiente local (XAMPP / WAMP / Laragon)

---

## Banco de Dados

O projeto utiliza MySQL para armazenamento dos dados.  
Os arquivos do banco foram gerados e utilizados via **phpMyAdmin em ambiente localhost**, com finalidade demonstrativa.

> Este projeto não possui foco em deploy em produção, sendo voltado para estudos e apresentação de conhecimento técnico.

---

## Como Executar o Projeto

1. Instale um servidor local (XAMPP, WAMP ou Laragon)
2. Coloque o projeto na pasta `htdocs` (ou equivalente)
3. Inicie o Apache e o MySQL
4. Importe o banco de dados via phpMyAdmin
5. Acesse o projeto pelo navegador: http://localhost/academia-web
